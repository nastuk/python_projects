## Описание проектов
Мини-проекты были выполнены в рамках обучения на программе ДПО "Азбука цифры. Профессия Python-программист".

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [1. Приложение для отображения курсов обмена криптовалют](https://github.com/nastuk/python_projects/blob/master/CryptoCurrencies.py) | В проекте разработано оконное приложение для отслеживания информации о курсах популярных крипловалют по отношению к основным международным валютам. Актуальные курсы валют загружаются с использованием API сервиса www.coingecko.com/en/api. | *requests, json, tkinter* |
| [2. Приложение для загрузки файлов в облако](https://github.com/nastuk/python_projects/blob/master/FilesIO.py) | В проекте разработано оконное приложение для загрузки файлов с компьютера на облачный сервер https://file.io. | *json, tkinter, pyperclip, os* |
| [3. Приложение таймер-напоминание](https://github.com/nastuk/python_projects/blob/master/Notification_upd.py) | В проекте разработано оконное приложение для таймера с напоминаниями, включая звук и текст напоминания. | *tkinter, datetime, time, pygame* |
